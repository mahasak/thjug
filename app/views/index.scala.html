THJUG is a Java & JVM user group held in Bangkok focusing on the JVM technology stack. Java & JVM is an Open Source technology that gained a lot of popularity since 1999, developed by James Gosling, Mike Sheridan, and Patrick Naughton in Sun Microsystems

Java & JVM technology is used by many of the biggest companies in the world because of its proven ability to enable better quality software. Want to learn the tech? Start here! We are very friendly and we help beginners get started and CTOs to get smarter. That's how the THJUG community works!
