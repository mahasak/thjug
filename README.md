THJUG.com
=================================
This website implement with Play Framework 2.3.9 + Scala.


```Instruction
brew install typesafe-activator sbt
activator ui
```


```Activator
activator clean
activator compile
activator test
activator run
activator dist
```


```Start
bin/thjug -Dhttp.port=80 -Djava.net.preferIPv4Stack=true & disown
```
